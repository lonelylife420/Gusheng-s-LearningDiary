[Leetcode : 69.x 的平方根](https://leetcode-cn.com/problems/sqrtx/)

我选择暴力算法从1开始 主要是它有个注意限制：不允许使用任何内置指数函数和算符，例如 pow(x, 0.5) 或者 x ** 0.5 。

```cpp 
class Solution {
public:
    int mySqrt(int x) {
        if (x == 0) return 0;
        for (long long i = 1; i <= (x + 1) / 2 ; i++) {
            if( i * i <= x && (i + 1L * (i + 1) > x) return i;
        }
        return -1;
    }
};
```
